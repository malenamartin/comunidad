# Propuesta UX/UI — Comunidad CMO (Fardo DS v1.4)

## Problemas detectados

1. Falta claridad de valor en la primera pantalla: no queda obvio qué logra un CMO en 5-15 minutos.
2. Exceso de ruido visual y baja jerarquía en elementos clave de acción.
3. Inconsistencia entre pantallas (temas oscuros y claros mezclados, estilos de componentes heterogéneos).
4. Panel admin disperso y sin narrativa operativa clara.

## Mejoras aplicadas en esta iteración

1. Base visual alineada a Fardo DS v1.4 (tokens semánticos, tipografía Satoshi, palette y bordes).
2. Feed con bloque "Playbook CMO" y quick actions para orientar decisiones.
3. Navegación y cards con contraste y legibilidad mejorados para lectura rápida.
4. Admin unificado con accesos a moderación, videos, betas y avatares.

## Próximas mejoras recomendadas

1. Dashboard inicial por rol (CMO, founder, admin) con KPIs y tareas sugeridas.
2. Feed con resumen semanal automático: "qué cambió", "qué hacer hoy", "riesgos".
3. Plantillas de post guiadas (benchmark insight, experimento, request de feedback).
4. Mejora de búsqueda y filtros por industria, país, seniority y objetivo.
5. Onboarding en 3 pasos con progress bar para activar al usuario en su primera sesión.
